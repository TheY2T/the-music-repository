## Goal

On electric bass, major scales are learned as **movable box shapes** rather than open-string
patterns — memorise the shape once and you can play any major scale just by sliding it to a new root.
This builds fretting-hand fingering, position sense, and the "one-finger-per-fret" discipline that
makes fast lines clean. (Standard 4-string tuning, low to high: `E A D G`.)

## How to do it

Use one finger per fret: **1** = index, **2** = middle, **3** = ring, **4** = pinky.

**One-octave shape (root on the E or A string, played with finger 2):**

```
root string : 1 (deg 1)   4 (deg 2)
next string  : 1 (deg 3)   2 (deg 4)   4 (deg 5)
next string  : 1 (deg 6)   3 (deg 7)   4 (deg 8)
```

Worked in **G major** with the root `G` at the E-string 3rd fret:

- `E` string: `G`(fr 3, finger 2), `A`(fr 5, finger 4)
- `A` string: `B`(fr 2, finger 1), `C`(fr 3, finger 2), `D`(fr 5, finger 4)
- `D` string: `E`(fr 2, finger 1), `F♯`(fr 4, finger 3), `G`(fr 5, finger 4)

Steps:

1. **Play it ascending and descending, slowly,** naming each note aloud.
2. Keep fingers close to the frets; press just behind the fret, not on top of it.
3. **Move the whole shape** up two frets and it becomes A major; slide it onto the A string and it
   becomes C major, D major, and so on. Nothing about the shape changes — only the root.
4. Anchor everything to a metronome at ♩ = 60, one note per beat, then two.

**Two octaves:** continue past the octave `G` across all four strings, or shift up a position — a
common approach is to play the one-octave box, then repeat it starting on the `A`-string root for the
upper octave.

## Common mistakes

- **Anchoring the thumb too hard** behind the neck — keep it relaxed and roughly opposite finger 2.
- **Reaching instead of shifting.** If a note feels like a stretch, move your whole hand position.
- **Buzzing notes** from pressing too far from the fret or not firmly enough.

## How to progress

Cycle the shape through every key around the fretboard, then add the two-octave version and increase
tempo on [/tools/metronome](/tools/metronome). Visualise the notes with the
[Fretboard tool](/tools/fretboard), then put the scale to work in
[Root–Fifth Bass Patterns](/bass-root-fifth-patterns).
