## Goal

Major-scale shapes turn the ukulele fretboard from a mystery into a map. Learn one shape and you can
play a scale in any key just by sliding it — the same movable idea guitarists and bassists use. This
builds finger accuracy, fretboard knowledge, and the melodic vocabulary behind riffs and solos.
(Standard re-entrant tuning, strings 4→1: `G C E A`.)

## How to do it

Start with the **open-position C major scale** — the clearest one to hear and see. Strings numbered
`4` (top `G`) to `1` (bottom `A`):

| String | Notes (fret) |
| --- | --- |
| C string (3) | C (open), D (fr 2) |
| E string (2) | E (open), F (fr 1), G (fr 3) |
| A string (1) | A (open), B (fr 2), C (fr 3) |

So the scale runs `C D E F G A B C` across the bottom three strings. See it on the neck — tap a note to hear it:

```embeds
[{ "tool": "fingering", "title": "C major on the ukulele", "caption": "The C major scale across the ukulele neck (g C E A), root C highlighted. Tap a note to hear it.", "instrument": "ukulele", "root": "C", "scale": "major" }]
```

Steps:

1. **Play it up and down slowly,** one finger per fret, saying the note names.
2. **Fret cleanly** just behind each fret so open and fretted notes ring equally.
3. Loop it until the sound of the major scale is locked in your ear.

**Making it movable (closed shape).** Replace the open strings with a fretted root so the shape can
slide. A one-octave closed major shape with the root on the C string (string 3) at fret `n`:

| String | Frets (scale degree) |
| --- | --- |
| C string (3) | n (deg 1), n+2 (deg 2), n+4 (deg 3) |
| E string (2) | n+1 (deg 4), n+3 (deg 5) |
| A string (1) | n (deg 6), n+2 (deg 7), n+3 (deg 8) |

Because there are no open strings, sliding the whole shape up two frets raises the whole scale a whole
step — move it and read off the new key. Practise to a click at ♩ = 60.

## Common mistakes

- **Muting the next string** by letting a fretting finger lean flat — keep fingertips arched.
- **Uneven volume between open and fretted notes** — press a touch firmer on fretted ones at first.
- **Forgetting where the root is.** Always know which note is "1"; that is what makes the shape movable.

## How to progress

Slide the closed shape into different keys, add a second octave, and speed up on
[/tools/metronome](/tools/metronome). Map the notes with the [Fretboard tool](/tools/fretboard), then
apply the scale melodically in [Aloha ʻOe](/aloha-oe-ukulele).
