The blues is a feel and a form. This path starts with the 12-bar structure, adds the scale that makes
it sing, then puts it all together with a left-hand bass line and a classic blues standard to play.

## Outcomes
- Play a 12-bar blues in multiple keys
- Improvise with the blues scale over the form
- Build a simple walking bass line

## Section: Putting It Together
- walking-bass-basics (note: A steady quarter-note bass turns changes into groove. skills: [walking-bass])
- handy-st-louis-blues (note: A real standard to apply everything to.)
