# Documentation

Docs are a **first-class deliverable of every phase** (see root `CLAUDE.md`).

- [`architecture/`](./architecture) — system + software/code architecture (Mermaid).
- [`adr/`](./adr) — Architecture Decision Records, one per significant choice.
- [`features/`](./features) — one doc per feature (`_template.md` to copy).
- [`conventions.md`](./conventions.md) — casing, naming, folder rules.
- [`content/`](./content) — content sourcing, licensing & attribution rules.
